# 🐍 经典贪食蛇 (Snake Game)

一个基于纯 HTML5 + Canvas 开发的现代化贪食蛇游戏。拥有丝滑的操控体验、完善的通关机制和本地排行榜系统。无需安装，打开浏览器即可游玩！

🔗 **在线试玩**: [点击这里开始挑战](https://kenvlezhangmin.github.io/snake-game/)
## ✨ 游戏特色

* **🏆 通关挑战机制**：不再是无尽枯燥的刷分，设有明确的通关目标！
    * **简单模式**：目标 50 个苹果（速度较慢，适合新手）
    * **困难模式**：目标 100 个苹果（标准速度，挑战手速）
    * **地狱模式**：目标 200 个苹果（极速挑战，反应极限）
* **📱 全平台适配**：
    * **PC 端**：支持键盘方向键操作。
    * **移动端**：完美支持触摸滑动操作，随时随地都能玩。
* **💾 本地数据存储**：
    * 利用浏览器 `LocalStorage` 技术，自动保存您的最佳战绩和历史记录。
    * 排行榜数据永久保存在您的设备上，不会丢失。
* **🎵 丰富的音效**：包含吃果子、撞墙失败以及通关胜利的专属音效和特效。
* **📊 数据管理**：支持将游戏记录导出为 JSON 文件备份。

## 🛠️ 技术栈

* **核心**：HTML5 Canvas
* **逻辑**：原生 JavaScript (ES6+)
* **样式**：CSS3 (Flexbox 布局)
* **存储**：Web LocalStorage API
* **部署**：GitHub Pages / Vercel / Netlify
